YT-Localizer (LiuSu)

YouTube video auto-localization desktop tool. Download, LLM-translate subtitles, burn bilingual Chinese captions, and produce ready-to-upload mp4 files with channel batch support.

Python PySide6 ffmpeg yt-dlp OpenAI API Firefox Deno

Overview

YT-Localizer (internally named LiuSu) is a one-click portable Windows desktop application that automates the full pipeline from YouTube video download to bilingual Chinese-captioned mp4 output. It leverages LLM APIs to intelligently translate and re-sentence subtitles, producing natural bilingual captions burned directly into the video.

The tool is designed for content creators and translators who need to localize English YouTube videos into Chinese at scale. It handles everything from subtitle extraction and ASR correction to font rendering and hardware-accelerated encoding, with no manual intervention required after initial configuration.

Key Features

  • One-click portable Windows desktop app built with PySide6 -- no installation required
  • Channel batch processing: paste a channel or playlist URL, select videos, and queue sequential processing
  • Smart subtitle mode with LLM-powered ASR correction, re-sentencing, and translation alongside strict 1:1 alignment mode
  • Beautiful bilingual subtitle styling using Source Han Sans with adaptive sizing
  • Hardware encoding auto-detection for accelerated video output
  • Firefox login integration for accessing premium and member-only content
  • Fully self-contained runtimes bundled in the distribution (Python, ffmpeg, fonts, Firefox Portable, Deno)

Tech Stack

The application bundles all dependencies into a portable distribution, eliminating the need for users to install any prerequisites. Technology choices prioritize cross-version Windows compatibility and offline operation.

  • UI Framework: PySide6 (Qt 6 for Python) -- native-feeling desktop interface
  • Video Pipeline: ffmpeg with hardware encoding detection (NVENC, QSV, AMF)
  • Download Engine: yt-dlp for video and subtitle acquisition
  • Translation: OpenAI-compatible LLM API (configurable endpoint, key, and model)
  • Browser Automation: Firefox Portable for authenticated cookie-based downloads
  • Subtitle Processing: Deno runtime for ASR correction and re-sentencing logic
  • Fonts: Source Han Sans bundled for consistent bilingual rendering

Usage

The application is distributed as a portable folder. No installation or system-level dependencies are required.

Quick Start

  1. Extract the distribution folder to any location
  2. Double-click yt-localizer.exe to launch
  3. Click the "AI - Not Configured" indicator in the toolbar, open Settings, and fill in your API endpoint, API key, and model name. Use "Test Connection" to verify.
  4. Paste a YouTube video or channel/playlist URL, select quality and subtitle options, then click "Start"

Output Structure

Processed videos and intermediate artifacts are stored in predictable locations relative to the application root:

text 
data/
  output/          # Final mp4 files with burned subtitles
  cache/
    <video-id>/   # Cached subtitles, intermediate SRT/ASS files

Known Limitations

  • Premium content: Member-only and premium videos use best-effort download with automatic fallback strategies
  • Cookie expiry: Firefox login cookies expire every 2-4 weeks and must be refreshed manually
  • Batch processing: Channel batch mode processes videos sequentially (one at a time), not in parallel
  • Channel listing: The channel video list is flat, showing only title and duration without publish dates

Development

Local development uses a Python virtual environment managed with uv. The application entry point is the app.main module.

Environment Setup

bash 
# Install dependencies into the project venv
uv pip install --python .venv\Scripts\python.exe -r requirements.lock

Running Locally

bash 
# Ensure UTF-8 output encoding
set PYTHONIOENCODING=utf-8

# Launch the application
.venv\Scripts\python.exe -m app.main
