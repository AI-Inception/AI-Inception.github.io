RAG-In-Action

GPT + RAG 实战经验与实现模式。面向生产级检索增强生成系统构建的公开参考合集。

Python LangChain LlamaIndex GPT

概述

RAG-In-Action 是一个精选参考合集，记录了构建检索增强生成（RAG）系统的真实工程经验。仓库沉淀了多个生产部署中积累的实用模式、工程决策和集成策略。

合集覆盖完整的 RAG 流水线——从文档摄入、分块策略，到向量检索、重排序，再到 LLM 响应生成。每篇文章针对生产中遇到的具体工程挑战，附带可运行代码和架构设计理由。

归档说明：本仓库最后活跃维护时间为 2024 年 6 月。如需最新 RAG 实践，请参考 LangChain 和 LlamaIndex 官方文档。此处记录的模式作为基础参考资料仍然有效。

  • 端到端 RAG 流水线实现，支持多种框架选择
  • 生产部署模式，包括 RAG-GPT 快速网站集成
  • 重排序策略，提升检索精度
  • 多供应商 LLM 集成（OpenAI、智谱 AI、DeepSeek、Ollama）
  • 企业级客服架构，配合领域专属知识库

技术栈

参考实现覆盖多种框架和供应商，反映了 2024 年初 RAG 生态的演进状况。

  • 编排框架：LangChain、LlamaIndex
  • 语言模型：OpenAI GPT-3.5/4、智谱 AI、DeepSeek、Ollama（本地）
  • 向量数据库：ChromaDB、FAISS、Milvus
  • 部署方案：RAG-GPT 框架、Docker、FastAPI
  • 编程语言：Python 3.10+

目录

仓库包含以下文章和实现指南：

RAG 技术深度解析

全面介绍如何使用检索增强生成构建下一代问答系统。涵盖理论基础、架构模式，以及不同分块和检索策略之间的取舍。

RAG-GPT：5 分钟部署 LLM

快速部署框架，为任意网站添加 LLM 驱动的问答能力。在单个可部署包中演示最小可行 RAG 流水线——文档摄入、向量索引和对话式检索。

利用重排序提升 RAG 效果

解决朴素向量检索的精度缺口。实现 Cross-Encoder 重排序以提升回答相关性，并提供集成重排序前后检索质量的基准对比。

LangChain 智能客服

以 RAG-GPT 为基础，结合 LangChain 编排构建生产级客服系统。涵盖对话记忆、上下文窗口管理以及向人工坐席的优雅降级。

智谱 AI 与 DeepSeek 集成

将 OpenAI Cookbook 的客服模式适配到国产 LLM 供应商。记录 API 兼容层、提示工程差异，以及跨供应商的成本性能权衡。

RAG-GPT 工程挑战

生产环境 RAG-GPT 部署的复盘与经验教训。涵盖文档更新流水线、Embedding 漂移、延迟优化，以及面向客户系统中对抗性查询的处理。

RAG-GPT 搭配 Ollama

使用 Ollama 进行本地推理，运行完整 RAG 流水线。演示敏感文档不离开本地网络的隐私保护部署方案，并针对消费级硬件进行性能调优。

OpenIM Bot：企业级智能客服

基于 LLM 与领域专属知识库的企业级智能客服架构。集成 OpenIM 多渠道消息，支持并发会话与会话隔离。

使用指南

克隆仓库并浏览各篇文章。每个子目录包含各自的 README，附带对应实现的环境配置说明。

bash 
git clone https://github.com/AI-Inception/RAG-In-Action.git
cd RAG-In-Action

环境配置

大多数实现需要 Python 3.10+ 和 OpenAI API 密钥。部分文章演示了无需 OpenAI 凭据的替代供应商方案。

bash 
# Create virtual environment
python -m venv .venv
source .venv/bin/activate  # or .venv\Scripts\activate on Windows

# Install dependencies for a specific article
pip install -r requirements.txt

# Set API keys
export OPENAI_API_KEY="sk-..."

运行 RAG-GPT

RAG-GPT 框架提供最快的可用 Demo 搭建路径。它将文档摄入、向量存储和聊天界面打包在单个服务中。

bash 
# Start RAG-GPT with Docker
docker compose up -d

# Or run directly
python main.py --docs ./your-documents/ --port 8080
